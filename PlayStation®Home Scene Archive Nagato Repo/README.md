# PlayStation-Home-Space-Archive
This repository will keep a log of all of the current spaces we have archived for the Destination Home Project. These spaces will include spaces/scenes from all regions of Home (SCEA, SCEE, SCEJ SCEA *Asia* & Debug/DEV) Note that this archive is purely for preservation & educational purposes.

Additional Information:

This archive will also include duplicate scenes with diffrent HDK versions spanning from 0.51 to 1.86. As well 1.1.0 SE (Scene Editor) to 1.3.0 SE.
